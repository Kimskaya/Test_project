Добавим новое предложение

  • 1. Добавить выбранный проект к себе как удаленный репозиторий с помощью функции Fork
  • 2. Уже со своего аккаунтра перенести проект на свою локальную версию с помощью команды git clone
  • 3. Внести изменения, которые необходимо и которые хотим превнести в чужой проект. Изменения стоит вносить на отдельной ветке и при отправке стоит пользоваться командой git push -u(или в полном варианте --set -upstream) origin название ветки
  • 4. Отправить измененную версию на свой удаленный репозиторий командой git push
  • 5. Со своего удаленного репозитория отправить свой измененный вариант владельцу репозитория, который мы изменяли с помощью команды compare&pull request (сравнить и отправить)
