# Основы работы с удаленным репозиторием
git clone — создание копии (удаленного) репозитория
git pull — забираем изменения из центрального репозитория
git push — вносим изменения в удаленный репозиторий
# Как сделать pull request
1. Делаем форк репозитория
2. делаем клон своей версии репозитория
3. создаем новую ветку и в нее вносим свои измения
4. фиксируем изменения (commit)
5. отправляем свою версию в свой гит хаб
6. на сайте гит хаб нажимаем кнопку pull request
